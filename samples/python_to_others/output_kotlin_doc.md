## Case sensitive
Kotlin is case sensitive

## Inline comment
Use //

## Block comment   
Use /* */ OR /** */

## Defining a function
```kotlin
fun myfunc(a: Int, b: Int) {
    val sum = a + b  
    println("sum: $sum")  
}
```

# Defining a for loop
```kotlin
for (i in 0 until 10) {  
    println(i)
}
```

# Defining if-else statement
```kotlin
val day = "Sunday"
if (day == "Sunday") {
    println("Sleep")
} else { 
    println("Also sleep")
}
```
